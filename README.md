# ДЗ2ТП

## Описание
Домашнее задание 2 по ООП и тестированию. Реализованы классы Ingredient, Recipe, DietaryRecipe, ShoppingList и тесты.

Установка зависимостей:
pip install -r requirements.txt

Запуск тестов:
pytest
